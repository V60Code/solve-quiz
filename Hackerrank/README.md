# Collection of Hackerrank Problem Solutions

This folder contains a collection of solution codes for various problems on the [HackerRank](https://www.hackerrank.com/) platform.

## Folder Structure

- **Algorithm**: This folder contains solutions for problems in the Algorithm category.
  - `#1- solve-me-first.md`: Solution for the "Solve Me First" problem.

## Contribution

If you want to contribute, please create a pull request with a new problem solution. Make sure to follow the existing folder structure.
