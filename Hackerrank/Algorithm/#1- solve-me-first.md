# Solve Me First

Complete the function `solveMeFirst` to compute the sum of two integers.

### Example

`a = 7`

`b = 3`

Return 10.

### Function Description

Complete the `solveMeFirst` function with the following parameter(s):

- `int a`: the first value
- `int b`: the second value

**Returns**

- `int`: the sum of a and b

### Constraints

1 ≤ a, b ≤ 1000

### Sample Input

```text
a = 2
b = 3
```

### How to Answer

just add value a and b then you got the answer :v
`return a + b`

### Code

```
def solveMeFirst(a,b):
    return a+b
	# Hint: Type return a+b below


num1 = int(input())
num2 = int(input())
res = solveMeFirst(num1,num2)
print(res)
```
